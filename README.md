#  Unity 2D Two-Player Arena Game

A local competitive 2-player arcade game built in Unity.
Two players control colored cubes inside a circular arena
and compete to collect objectives while avoiding penalties.

##  Gameplay Overview
- Two players play simultaneously
- Each player controls a cube of a different color
- Players move inside a circular ring (arena)
- Objective: collect white balls faster than the opponent


##  White Balls (Scoring)
- White balls spawn randomly
- Collecting white balls increases score
- The player who collects all / maximum white balls first wins


##  Red Balls (Penalty)
- Red balls spawn randomly
- Touching a red ball freezes the player for 1â€“2 seconds
- Creates risk and strategic movement

##  Power-Up System
- After collecting a certain number of white balls:
  - Player size increases
  - Player movement speed increases
- Encourages aggressive and strategic gameplay


##  Key Features
- Local 2-player input handling
- Random object spawning
- Power-up & debuff mechanics
- Physics-based movement
- Win condition logic


##  Tools & Technologies
- Unity (2D)
- C#
- Rigidbody2D & Collider2D
- Input Manager / Input System
